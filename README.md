# Staff Management System

## Tech Stack
- ASP.NET Core 8
- SQL Server
- Entity Framework Core
- ASP.NET Identity

## Default Credentials
Admin:
Email: admin@staffsystem.com
Password: Admin@123

## Run Instructions

1. Install .NET 8 SDK
2. Update connection string in appsettings.json
3. Run:
   dotnet restore
   dotnet ef database update
   dotnet run

Open https://localhost:5001
